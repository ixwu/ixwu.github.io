使用Pelican+Github Pages搭建静态博客

Pelican是一款用Python编写的静态网站生成器。可以把reStructuredText或者Markdown编写的文本变成html页面，然后托管到Github pages上，便可轻松实现一个静态博客。

比较常见的静态网站生成工具还有：Hexo、Jekll、Hugo等。

其好处很明显，不需要自己买服务器了，直接托管到github上，而且占用资源少，又不用数据库，速度自然快

接下来跟着我一块配置吧。

配置github pages

这段不再详说了，前提你需要有一个GitHub账号，然后创建一个仓库，命名方式为：用户名.github.io。

我在GitHub创建一个仓库，命名为：xwlearn.github.io

下载Pelican

pip install pelican ghp-import markdown

由于静态博客都是markdown文章，所以需要下载python的markdown模块，ghp-import用于将
